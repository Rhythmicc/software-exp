# software-exp

## Install

- java: 11.0.10
- Vue ^2.9
- Maven

```shell
chmod +x install.sh
chmod +x backend.sh
chmod +x frontend.sh

./install.sh bf # which means install backend & frontend
```

## Run

```shell
./backend.sh  # Start up backend
./frontend.sh # Start up frontend
```

## ENV

- OS: Linux & Mac & ... (if you want to use Windows, you should install by yourself)
